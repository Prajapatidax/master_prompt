## ⚡ Available Prompts

### 1. 💻 Codex Prompt (Fast UI Upgrade)

Best for:

* Quick improvements
* Direct code generation

```text
You are a senior frontend engineer and UI/UX designer.

TASK:
Enhance and modernize the following frontend code while keeping functionality intact.

GOALS:
- Improve UI aesthetics (modern, clean, premium feel)
- Improve UX (better spacing, hierarchy, readability)
- Make it responsive for mobile, tablet, desktop
- Improve performance and maintainability

CONTEXT:
- Language/Framework: [INSERT]
- Purpose: [INSERT]
- Target users: [INSERT]
- Design style: [INSERT]

INSTRUCTIONS:
- Improve design (spacing, typography, colors)
- Enhance UX and layout flow
- Make responsive (mobile-first)
- Refactor clean code
- Add animations, dark mode (optional)

CRITICAL STEP:
Analyze UI and redesign it to be 10x better.

OUTPUT:
- Explanation
- Full improved code

INPUT CODE:
[PASTE HERE]
```
